## About the Project

**Culture Event** is a full-stack web application for browsing and managing cultural events.  
The platform allows visitors to view the event catalog, open detailed event pages, read news, leave comments, send messages through the contact form, register accounts, log in, manage a profile, add events to favorites, and book tickets.

The project includes:

- a **client side** built with **React + Vite**
- a **server side** built with **Node.js + Express**
- a **MongoDB database**
- **JWT authentication**
- **role-based access** for regular users and administrators

The application is designed as an interactive cultural event portal where users can not only browse events, but also interact with the platform through booking, commenting, favorites, and profile customization.

---

## Main Goals of the Project

The main purpose of the project is to develop a responsive and functional web application that demonstrates:

- modern frontend development with React
- routing in a Single Page Application
- client-server interaction through REST API
- authentication and authorization
- CRUD operations with database entities
- profile and user data management
- protected routes for private and admin-only pages
- practical organization of a full-stack application structure

## Key Features

- Viewing all available cultural events
- Filtering events by category
- Searching events by title
- Viewing full event details
- Booking tickets for events
- Cancelling booked tickets
- Adding and removing events from favorites
